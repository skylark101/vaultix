# Vaultix

**Vaultix** is a personal financial command center to track and manage everything you own — from investments and insurance to real-world assets — in one clean, fast, and private interface.

> Everything you own. One place.

---

## 🚀 Features

- 🔐 **Authentication**
  - Secure login & signup (JWT-based)

- 📊 **Asset Management**
  - Track any type of asset (FD, Insurance, Real Estate, etc.)
  - Flexible schema using custom fields

- 🧠 **Simple Dashboard**
  - Total invested amount
  - Asset count
  - Recently added items

- 🔎 **Search & Filter**
  - Filter by asset type
  - Quick search by name

- 🗂️ **Extensible Design**
  - JSON-based custom fields for flexibility
  - Easy to support new asset types

- 🎨 **Modern UI**
  - Dark theme
  - Minimal, fast, distraction-free interface
